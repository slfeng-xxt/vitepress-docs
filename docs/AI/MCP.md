# MCP

## 1. MCP是什么?

:::tip
让ai有了四肢可以干活
:::

- [mcp](https://modelcontextprotocol.io/docs/getting-started/intro)

### 1.1 MCP host

桌面应用： cursor chatgpt claude  trae

### 1.2 MCP client（ai agent）

host里面的自然语言翻译工具，讲自然语言转为MCP server听的懂的语言格式

### 1.3 MCP server

实际执行干活的插件， 例如： gmail server / excel server / github server/ unity server

### 1.4 MCP 协议

一套规范，确保上述 host client server 之间可以通信
