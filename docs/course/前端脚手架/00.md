# 流程详解

## 开发流程

- 创建npm项目
- 创建脚手架入口文件，第一行添加（#!/usr/bin/env node）
- 配置package.json,添加bin属性
- 编写脚手架代码
- 将脚手架发布到npm

## 使用流程

- 安装脚手架

```js
npm install -g your-cli
```

- 使用脚手架

```js
your-cli
```
