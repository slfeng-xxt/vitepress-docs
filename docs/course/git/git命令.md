# git命令

## git init

初始化git仓库

## git add

将文件添加到暂存区

## git commit

将暂存区的文件提交到仓库

## git status

查看git状态

## git log

查看git提交历史

## git reset

重置git状态

## git branch

查看、创建、删除分支

## git checkout

切换分支

## git merge

合并分支

## git rebase

变基

## git remote

查看、添加、删除远程仓库

## git pull

拉取远程仓库代码

## git push

推送代码到远程仓库
