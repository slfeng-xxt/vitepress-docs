# 智慧储能系统(EMS)

:::tip
本项目为智慧储能系统，主要功能包括储能设备的监控、数据采集、故障诊断、优化调度等。
:::

## 1.项目搭建

- 参考：<https://github.com/youlaitech/vue3-element-admin>
- 文档：<https://vue3-element-admin-site.midfar.com/zh/guide/#%E5%8A%9F%E8%83%BD>

## 2.特色功能

- 添加 `agent skills`, vue-best-practices
- element-plus主题样式定制
- 接口api的柯里化
- 动态路由生成
- 设备列表不同状态查看页面的参数化（动态组件+通用接口配置）
